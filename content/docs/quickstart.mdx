# Quick Start

Get started with BluesMinds in under 5 minutes.

## Prerequisites

- Node.js 18+ (or any runtime that supports `fetch`)
- A BluesMinds API key (create one in `/dashboard/keys`)
- Set `BLUESMINDS_API_KEY` in your environment

```bash
export BLUESMINDS_API_KEY=bm_live_your_api_key
```

## Install the SDK

Install the BluesMinds SDK using your preferred package manager:

```bash
npm install @bluesminds/sdk
# or
yarn add @bluesminds/sdk
# or
pnpm add @bluesminds/sdk
```

## Initialize the client

Create a client instance with your API key:

```typescript
import { BluesMinds } from '@bluesminds/sdk';

const client = new BluesMinds({
  apiKey: process.env.BLUESMINDS_API_KEY
});
```

## Make your first request

Send a chat completion request to any supported model:

```typescript
const response = await client.chat.completions.create({
  model: 'gpt-4',
  messages: [
    { role: 'user', content: 'Hello! How are you?' }
  ]
});

console.log(response.choices[0].message.content);
```

## Use different providers

Switch providers by changing the model name. To see what is available on your gateway, call the [`/v1/models` endpoint](/docs/api/models).

```typescript
// OpenAI
await client.chat.completions.create({
  model: 'gpt-4',
  messages: [{ role: 'user', content: 'Hello!' }]
});

// Anthropic (Latest Claude 4.5)
await client.chat.completions.create({
  model: 'claude-sonnet-4.5',
  messages: [{ role: 'user', content: 'Hello!' }]
});

// Google
await client.chat.completions.create({
  model: 'gemini-pro',
  messages: [{ role: 'user', content: 'Hello!' }]
});
```

## Error handling

Handle errors with a `try/catch` and branch on status codes when possible:

```typescript
try {
  const response = await client.chat.completions.create({
    model: 'gpt-4',
    messages: [{ role: 'user', content: 'Hello!' }]
  });
  console.log(response);
} catch (error) {
  if (error.code === 'rate_limit_exceeded') {
    console.error('Rate limit exceeded');
  } else if (error.code === 'authentication_error') {
    console.error('Invalid API key');
  } else {
    console.error('An error occurred:', error.message);
  }
}
```

## Troubleshooting

### 401 / authentication errors

- Confirm `BLUESMINDS_API_KEY` is set in the environment where your code runs.
- Ensure the header is `Authorization: Bearer <key>` (the SDK does this automatically).

### Model not found

- Call [`/v1/models`](/docs/api/models) and use an `id` from the response.
- If self-hosting, ensure providers and model mappings are configured in the admin UI.
